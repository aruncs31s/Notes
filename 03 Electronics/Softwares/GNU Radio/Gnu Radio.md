## Tutorial
- Integer data type `int(10e3) -> 10000` , `int(3.2222) -> 3`
- `this is string`
- `.25` is `250m` -> converted to milli
- Lists -> [1,2,4,'k',4]
- Tuples -> (1,2,3,4)

```python
>>> (1,2,3)[:]
(1, 2, 3)
>>> (1,2,3,(2,3,4))
(1, 2, 3, (2, 3, 4))
>>> (1,2,3,(2,3,4))[3]
(2, 3, 4)
>>> (1,2,3,(2,3,4))[3][2]
4
>>> (1,2,3,(2,3,4))[3][:]
(2, 3, 4)
>>>
```
