## Quick Reference

### Status Codes

| Range | Meaning |
|-------|--------|
| 2xx | Success |
| 3xx | Redirection |
| 4xx | Client error |
| 5xx | Server error |

### HTTP Methods

| Method | CRUD | Safe | Idempotent |
|--------|------|------|----------|
| GET | Read | Yes | Yes |
| POST | Create | No | No |
| PUT | Update | No | Yes |
| PATCH | Update | No | No |
| DELETE | Delete | No | Yes |
