# Variables & Data Types

Learn about Python's basic data types.

## Numbers

```python
x = 5           # int
y = 3.14        # float
```

## Strings

```python
name = "pytest"  # str
```

## Booleans

```python
is_active = True
```

## Lists

```python
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", True]
```

## Dictionaries

```python
user = {
    "name": "Ganga V",
    "email": "gangav@example.com",
    "age": 30
}
```

## Sets

```python
unique_numbers = {1, 2, 3}
```

## Tuples

Tuples are immutable (cannot be changed):

```python
coordinates = (10, 20)
```
