# Git Class

---

- Git is a free, open source **version control tool** 
- GitHub is a pay-for-use online service built to run Git in the cloud
---
## Installation
- Linux/Ubuntu 

```bash
sudo apt install git

```

- Arch

```bash
sudo pacman -S git
yay -S git

```

- Fedora

```bash
sudo dnf install git 

```
