## Install 

```bash
sudo pacman -S mysql

```

```bash
sudo mysqld --initialize --user=mysql --basedir=/usr --datadir=/var/lib/mysql

```

## Setup 
@initial

```bash
sudo mysqld_safe --skip-grant-tables --skip-networking &
mysql -u root

```

```mysql
FLUSH PRIVILEGES;
ALTER USER 'root'@'localhost' IDENTIFIED BY 'your_new_password';
FLUSH PRIVILEGES;
EXIT;

```
